Выпуск #1 Тема Роблокс
|Главная|

«Spectre 2» — игра, вдохновленная фазмофобией. Суть игры заключается в том, чтобы определить тип призрака по 7 различным характеристикам и выбрать только 3 из них. Призрак может оставлять отпечатки пальцев, совершать паранормальные движения, оставлять сферы, которые можно увидеть через специальные очки, и многое другое.
